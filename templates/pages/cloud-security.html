Best-in-class security

Focus on your business, your data is yours

Focus on your business, your data is yours. Existing cloud such as Dropbox and Microsoft One Drive are today’s ultimate tools for collaboration and synchronization.

Because Tanker focuses on encryption only, it simply plugs into your current favorite cloud solution and adds that missing security feature you need to secure your business data:

  • Full end-to-end encryption (Elliptic Curve - secp256r1)
  • Digital signatures (ECDSA)
  • Automatic key sharing
  • Company master key for resilience
  • Administrator user management console with a security perspective

Smart end-to-end encryption

Works in harmony with the best cloud solutions

Your favorite cloud provider uses a large set of "tricks" to make your life easier: partial synchronization, LAN Sync, fast synchronization algorithms, Share...

Applying dummy encryption would breakdown all these features and at the same time destroy your productivity. To avoid this, Tanker uses its exclusive Forrest encryption format and network drive emulation:

  • Use AES256-GCM chunk encryption for partial updates and reads
  • Use ChaCha20 for local cache on large files and fast diff computations
  • Use network for fast key transfers among users
  • Use Merkle trees for fast file integrity checks and signatures
  • Emulate a WebDAV server for transparent file operations and full compatibility with your day-to-day tools

Seamless key management

Own your encryption key = own your data

Tanker is a zero-knowledge solution providing you with complete encryption: only you have the keys to unlock your files.
Each employee has its own local private key, never transmitted on the internet
Your company has a master encryption and signature key to recover lost accounts.
Document keys are never sent in clear form.
Tanker server only hold public keys and relies on TrustChainTM for authenticity.
To sum up, the only thing we hold are: encrypted document keys we can’t decrypt and public keys we can’t alter.

Open design & open source based

The more you know, the more you trust

Not everyone is an expert in cyber security and cryptography. Some of you will simply trust us, others will want to know more. Because security with Tanker is invisible, using Tanker will give you no clue on "how it works". Tanker uses the Open-Source (simplified-BSD) cryptographic library Botan for all its low level operations.
